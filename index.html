Ali

Accessible Learning Interactions

Ali Interactions is a growing suite of UI interactions for eLearning courseware environments that can use JavaScript, CSS and HTML5 to create rich, interactive content.

Minimally styled and patterned after the components in Twitter Bootstrap, Ali learning interactions are are easy to use and require only jQuery and a basic knowledge of HTML.

The components are compatible back to Internet Explorer 10, so your learners should have few issues making use of the content.

Developers who require more control to customize the build of the components will find use a modern front-end development stack that uses Grunt, SCSS with AutoPrefixer, jsHint and UglifyJS.

Future builds of the project will remove jQuery, migrate to Browserfy & ESlint, with ES6/Babel as the language of choice. In addition, future builds will feature opt-in build parameters that allow you to customize the JavaScript and CSS components used per project.
