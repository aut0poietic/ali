Ali

Accessible Learning Interactions

Ali Interactions is a growing suite of UI interactions for eLearning courseware environments that can use JavaScript, CSS and HTML5 to create rich, interactive content.

Minimally styled and patterned after the components in Twitter Bootstrap, Ali learning interactions are are easy to use and require only jQuery and a basic knowledge of HTML.

The components are compatible back to Internet Explorer 10, so your learners should have few issues making use of the content.

Developers who require more control to customize the build of the components will find use a modern front-end development stack that uses Grunt, SCSS with AutoPrefixer, jsHint and UglifyJS.

Future builds of the project will remove jQuery, migrate to Browserfy & ESlint, with ES6/Babel as the language of choice. In addition, future builds will feature opt-in build parameters that allow you to customize the JavaScript and CSS components used per project.

Accessible & Responsive

Out-of-the-box All Ali components are accessible and responsive. Full-stop. No caveats, no excuses.

Ali requires JavaScript and utilizes ARIA roles, states and properties properly communicate with assistive technology. Every component works with keyboard as well as the mouse and there are no overridden swipe gestures to break mobile screen readers.

The components are only minimally styled, allowing you to enhance them any way you see fit.

That said, they're not magic.

There's nothing within the components that can make an inaccessible course framework accessible. There's nothing stopping someone from adding inaccessible contrasting styles to the controls, hiding the focus indicator or adding additional code that harms accessibility, etc.

Project Ali

The Learning components are a part of a larger initiative to update the eLearning products created by the team behind the University of Oklahoma's Center for Public Management.

Additional tools (currently in development) include a go(-lang) based "static site generator" capable of producing SCORM/xAPI courses, grunt automation for scaffolding and rapid project development, and a course wire-framing and feedback system built into the WordPress publishing system.
