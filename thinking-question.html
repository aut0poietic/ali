Thinking Question

Why use a Thinking Question?

  • You have a question or scenario you want the user to compose a long-form answer in order to consider the concepts involved.
  • You do not actually want to save the user's response.

Why not a Thinking Question?

  • You need to evaluate the user's response.
  • The question has a correct and/or incorrect answer.

Format (mustache.js)

The basic structure of the control is represented as a mustache.js template.


						<form id="{{id}}" class="thinking-question" data-ali="thinking-question">
							<div class="question-wrapper">
								<div class="question">{{{stem}}}</div>
								<textarea></textarea>
								<div class="submit-row">
									<button class="submit" aria-label="Submit">Submit</button>
								</div>
							</div>
							<div class="feedback-content">
								{{{feedback}}}
							</div>
						</form>

Example

An individual's construction of social reality, not the objective input, may dictate their behaviour in the social world.

Take a moment and consider how your biases affect your social interactions and then explain in a paragraph or two how you believe you might manage the bias you experience — or eliminate them entirely.

Cognitive Bias Modification

Cognitive bias modification refers to the process of modifying cognitive biases in healthy people and also refers to a growing area of psychological (non-pharmaceutical) therapies for anxiety, depression and addiction called cognitive bias modification therapy (CBMT).

CBMT is sub-group of therapies within a growing area of psychological therapies based on modifying cognitive processes with or without accompanying medication and talk therapy, sometimes referred to as applied cognitive processing therapies (ACPT). Although cognitive bias modification can refer to modifying cognitive processes in healthy individuals, CBMT is a growing area of evidence-based psychological therapy, in which cognitive processes are modified to relieve suffering from serious depression,anxiety, and addiction.

CBMT techniques are technology assisted therapies that are delivered via a computer with or without clinician support. CBM combines evidence and theory from the cognitive model of anxiety, cognitive neuroscience, and attentional models.

Event Model

Event: ali:complete

Dispatched when all items have been completed.


						$( document ).on( 'ali:complete', $( 'dl.accordion' ),
							function ( e, data ) {
								// e: Object, the original event object
								// data: Object interaction event objec
								console.log( e, data );
							}
						);

Data Object

Passed with the event. Note the not all items will be populated for all events.

For example: latency has no meaning for an ali:itemSelected event unless the interaction was completed previously.


						data = {
							description       : {string}     // The description of the current interaction
							id                : {string ID)  // The id of the interactions main element
							latency           : {int}        // Milliseconds the user spent viewing the interaction
							result            : {string}     // Status of the element, either "correct" or "incorrect"
							start             : {int}        // Time the user was first shown the interaction, in milliseconds
							type              : {string}     // SCORM/xAPI interaction type
							correct_responses : {array}      // An array of the responses in the correct order as text
							learner_response  : {array}      // An array of the responses in the order chosen by the user as text
						};
